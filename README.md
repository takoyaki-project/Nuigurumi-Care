## 🌟 これは何？

**ぬいぐるみおせわ魔法帳** は、だいすきなぬいぐるみのお世話をして、なかよし度を上げていく子ども向けアプリです。

ごはん・おふろ・あそび・おさんぽ・ねんね。お世話をするたびにぬいぐるみが反応してくれて、レベルが上がるとアイテムがもらえます。ゆめかわ×ちょい小悪魔テイスト。

## 🚀 公開URL

[▶ アプリを開く](https://codepen.io/dfbitwqs-the-animator/full/XJNoXvv)

## ✨ できること

* ぬいぐるみの写真を登録して、名前をつけられる
* 5種類のお世話ができて、ぬいぐるみが表情とセリフで反応する
* ぬいぐるみの「おねがい」を叶えるとボーナス
* なかよし度が満タンになるとレベルアップ（みならい→だいすきパートナーの5段階）
* レベルアップでアイテム（リボン・ティアラなど7種）をゲット
* ときどきレアリアクション（にじいろまほう✨）が発生
* レベル・アイテム・きろく・なまえ・しゃしんは自動保存（v2で追加）

## 📱 使い方

1. ぬいぐるみの写真を登録して、なまえをつけます。
2. お世話ボタンをタップして、なかよし度を上げます。
3. レベルアップとアイテム集めを楽しみます。

## 🛠 使用技術

* HTML / CSS / JavaScript

* CSSアニメーション（星空・ハート・キラキラ演出）
* Google Fonts: Zen Maru Gothic
* CodePen

## 📝 制作メモ

娘のために作ったアプリです。「お世話するとちゃんと反応が返ってくる」ことを大事にして、表情・セリフ・エフェクトの組み合わせをたくさん用意しました。当初は保存機能がなく、閉じるとレベルが消えてしまっていたのですが、v2で自動保存に対応しました。
